Forest Clicker

First project in TypeScript and Next.JS even though I had very little knowledge of JavaScript and none of React. So I made a lot of mistakes but the application is functional.

It's a little clicker type game inspired by, for example, "Cookie clicker". The game focuses on the theme of reforestation with the aim of raising awareness among young people about deforestation. The goal is to click on the tree which earns us 1 point for each click. With these points we can buy improvements that allow us to gain, for example, 1 point per second. The more you buy, the more the improvements cost but the more points they earn per second. There are four types of improvements ranging from weakest to strongest. There is a description of our forest allowing, by taking the maximum number of trees that we had, to tell us its size in hectares as well as the number of KG of CO2 that it was able to capture per year.

As I said at the beginning, I made several code errors, particularly when using React. I plan to resume the project soon to resolve these issues and improve the game.
